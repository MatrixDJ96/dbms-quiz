# Maturità DBMS Chaos Mode

Progetto React/Vite compilabile e condivisibile.

## Requisiti

- Node.js recente compatibile con Vite
- npm

## Avvio in sviluppo

```bash
npm install
npm run dev
```

## Build condivisibile

```bash
npm run build
```

Il risultato finisce nella cartella `dist/`. Puoi caricare `dist/` su un hosting statico o comprimerla e condividerla.

## Preview locale della build

```bash
npm run preview
```

## Note

- App React con Canvas animato.
- Nessuna dipendenza grafica esterna.
- Audio sintetico via Web Audio API.
- Timer adattivo in base a lunghezza, difficoltà e complessità tecnica delle domande.
